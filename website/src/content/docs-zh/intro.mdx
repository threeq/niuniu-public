牛牛是一个**目标驱动的本地优先 AI 开发工作站**：把你想做的事写成 issue，多个 AI Agent 自主规划 → 执行 → 验证 → 收尾，跨项目、跨仓库并行推进。

## 它解决什么

传统 AI 编码助手是"一对一"工作模式——一个会话只能处理一个任务，过程中还要不停回头确认细节。当你同时维护 3 个项目、5 个 issue 时，这种"陪聊式"协作很快就会成为瓶颈。

牛牛把工作单位从"一次对话"升级为"一个目标"：每个 workspace 对应一个 issue，自带独立的 git worktree、独立的 Claude 会话、独立的命令行环境；agent 在这个隔离环境里自动闭环完成目标，只在真正阻塞时回到你这里确认。多个 workspace 并行运行，互不干扰。

## 两种交付形态

- **Personal** — 桌面应用，单机本地运行，三平台二进制。
- **团队私有版** — 通过 docker compose 在自有基础设施上部署，使用 PostgreSQL，私有数据，支持组织、成员、审计。

详细安装请见 [安装指南](/docs/install/personal)。
